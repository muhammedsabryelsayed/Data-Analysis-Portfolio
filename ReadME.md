**Amazon Sales Analysis Dashboard (2024–2026)**

**Project Overview**

This project presents a detailed sales analysis of 1,000 Amazon sales records collected across the years 2024, 2025, and 2026. The purpose of this analysis is to identify sales trends, customer behavior, product performance, and business insights using Excel-based data analysis techniques.
The project focuses on transforming raw sales data into meaningful insights through formulas, dashboards, automation, and data visualization,The project consists of an Amazon sales dataset along with additional randomly generated analytical data for training and practice purposes.
These datasets are used to simulate a real-world business environment and apply data analysis techniques to build an interactive dashboard.


---

**Objectives**

Analyze sales performance across multiple years.

Identify high-performing products and categories.

Track customer purchasing behavior.

Compare yearly sales growth and trends.

Build an interactive Excel dashboard for clear business insights.
